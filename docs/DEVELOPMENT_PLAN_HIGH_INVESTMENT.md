#### 技术架构

**存储方案**：
```typescript
interface GameReplay {
  id: string;
  roomId: string;
  script: string;
  startTime: number;
  endTime: number;
  events: GameEvent[];  // 压缩后的事件流
  result: 'good' | 'evil' | 'tie';
  players: PlayerSummary[];
}

interface GameEvent {
  timestamp: number;
  type: 'seat' | 'assign' | 'night' | 'vote' | 'death' | 'win';
  data: any;
}
```

**存储位置**：
- **LocalStorage**：最近 10 场（不依赖 Supabase）
- **Supabase**：云端无限存储（需要项目恢复）
- **导出 JSON**：本地保存
